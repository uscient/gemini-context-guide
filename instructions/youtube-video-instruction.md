# YouTube / Video Instruction

Use this when asking about a video, page, transcript, or link.

```markdown
For this video/page question, use only the specific video, page, link, transcript, or browser tab I provide.

Do not use my YouTube History, Chrome history, Search history, previous chats, saved information, Notebooks, projects, or connected apps unless I clearly ask.

If the video/page context is unavailable, say so. Do not fill gaps from unrelated information.

At the end, include:

## Information used
List what you used.

## Scope check
Say one of:
- provided source only
- current chat only
- I may have used outside information
- I cannot verify
```
