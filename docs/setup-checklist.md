# Setup Checklist

Use this checklist before relying on Gemini as a personal workspace.

## Account and settings

- [ ] Confirm which Google account you are using.
- [ ] Review Gemini activity / history / personalization settings.
- [ ] Review saved information or saved preferences.
- [ ] Review whether past chats may be used for personalization.
- [ ] Review connected app access, such as Gmail, Drive, Docs, Calendar, YouTube, or Photos.
- [ ] Review Gemini in Chrome settings, especially page or tab sharing behavior.
- [ ] Set retention options deliberately if the product offers them.

## Global instruction

- [ ] Add a short one-off chat containment instruction.
- [ ] Avoid putting project-specific rules in global instructions.
- [ ] Avoid private facts, legal strategy, or business doctrine in global instructions.

## Notebook setup

For each important Notebook:

- [ ] Add a Notebook boundary instruction.
- [ ] Keep sources relevant to that Notebook only.
- [ ] Remove or split mixed-project sources.
- [ ] Add domain-specific rules only where they belong.
- [ ] Run a Notebook boundary test.

## Routine use

- [ ] For important answers, require an “Information used” section.
- [ ] For important answers, require a “Scope check” section.
- [ ] If an answer seems to use outside information, run the source audit prompt.
- [ ] If a one-off chat becomes project-relevant, move it into the right Notebook and run a scope check.
